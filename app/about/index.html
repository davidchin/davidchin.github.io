Summary

I am David Chin, a developer specialising in front-end web and native application development. I am experienced in using JavaScript to write dynamic client-side applications interacting with JSON APIs. I have worked with some amazing teams on many high-traffic projects for well-known brands, helping them to resolve many complex technical issues.

Currently, AngularJS is my preferred client-side MVC framework as it is highly extensible and good at working with dynamic data. When writing CSS and HTML, I prefer using pre-processors such as SASS and HAML respectively due to their cleaner syntax.

I also have experience in writing native iOS applications using Swift, that run fast and memory efficient. I have a strong understanding of Ruby on Rails, having worked on various CMS and APIs.

I always test my code if I can, using BDD testing frameworks such as Jasmine to help write clean unit and E2E tests. Having worked at an agile development agency for many years, I am familiar with agile principles and continuous integration workflow.

I am good at evaluating technical approaches by considering the constraints and goals of a sprint. I am comfortable in engaging code review session with other team members.

I enjoy new challenges and learn about the best practices of modern application development.
