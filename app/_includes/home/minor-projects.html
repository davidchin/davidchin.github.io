Angular Endless Scroll

Angular Endless Scroll is an open-source AngularJS directive, developed to help people implement infinite scrolling UI. It attempts to address the memory issue of having too many DOM elements as the page gets longer, and allow deep-linking to paginated pages.

TinyPantry

TinyPantry is a recipe website built with AngularJS and Rails. It has an authentication system allowing users to save their favourite recipes. Recipes are scraped and categorised daily by a scheduled task - the frontend simply renders the data fetched from a versioned API.

Neuplace

Neuplace is a native iOS app showcasing curated images of interior designs. It is integrated with Tumblr JSON API, featuring an endless collection of images which users can bookmark and share. Dynamic contents and static assets are cached to reduce latency.
