Personal Projects

Here are some projects I worked on in my spare time.

Endless Scroll

Angular Endless Scroll is a directive used to implement infinite scrolling UI. As a page gets longer, it manages the number of elements in DOM tree, and allows deep-linking to paginated pages.

View project

TinyPantry

TinyPantry is a recipe website built with AngularJS and Rails. Recipes are indexed and categorised daily by a scheduled task - AngularJS is used to render the data fetched from a versioned API.

View project

Neuplace

Neuplace is a native iOS app showcasing curated images of interior designs. It is integrated with Tumblr JSON API, featuring an endless collection of images which users can bookmark and share.
