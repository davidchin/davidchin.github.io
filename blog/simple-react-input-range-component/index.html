Simple React Input Range component

react / 30 September 2015 / 0 Comments

Recently, I needed a component for accepting a range of numeric values inside a form. I couldnâ€™t just use a native input range slider as it only accepts a single value. So I thought I might just write one myself. I decided to release it as a standalone package. You can check it out on Github or try it in your project - npm install react-input-range.

Background

react-input-range allows users to input a range of numeric values or a single value within a predefined range. By default, basic styles are applied, but can be overridden easily.

Example

You can see a simple example below. For more detail, please refer to the README.

import React from 'react';
import InputRange from 'react-input-range';

const values = {
  min: 2,
  max: 10
};

function onChange(component, values) {
  console.log(values);
}

React.render(
  <InputRange maxValue={20} minValue={0} values={values} onChange={onChange} />,
  document.getElementById('input-range')
);
